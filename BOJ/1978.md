# [1978] 소수 찾기

## 1. 문제 분석

- **난이도:** Bronze II
- **링크:** [문제 링크](https://www.acmicpc.net/problem/1978)

### 접근 방법 (Thinking Process)

1. 1은 소수도 합성수도 아님.
2. 반복 돌면서 인덱스의 제곱이 대상 수 보다 커지는 경우 전까지 체크.
3. 나누어 떨어지고, 대상 수와 같지 않은 경우 합성수로 판단.

## 2. 코드 설계 (Pseudo Code)

```text
N 입력
N번 반복
1. 수 입력
2. 1인 경우 합성수 카운트 증가
3. 2 이상이면, 2부터 1씩 증가 시키며 제곱이 대상 수 보다 작을 경우 까지 반복
    1. 대상 수가 인덱스로 나누어 떨이지면서 대상 수와 같지 않는 경우 합성수 카운트 증가 -> break
N-c 출력
```

## 5. 최종 코드

```cpp
#include <iostream>

int main(void) {
    int N = 0;
    std::cin >> N;
    int c = 0;
    for(int i=0; i < N; i++) {
        int foo = 0;
        std::cin >> foo;
        if(foo==1) c++;
        if(foo>=2) {
            for(int bar=2; bar*bar<=foo; bar++){
                if(foo%bar == 0 && bar!=foo) {
                    c++;
                    break;
                }
            }
        }
    }
    std::cout << N-c << std::endl;
    return 0;
}
```
