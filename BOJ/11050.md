## 5. 최종 코드

```cpp
#include <iostream>
using namespace std;

int factorial(int num) {
    if(num <= 1) return 1;
    return num * factorial(num-1);
}

int main (void) {
    int n = 0, k = 0;
    cin >> n >> k;
    cout << factorial(n) / (factorial(n-k) * factorial(k)) << '\n';
}
```
