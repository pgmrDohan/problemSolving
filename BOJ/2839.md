# [2839] 설탕 배달

## 1. 문제 분석

- **난이도:** Silver IV
- **링크:** [문제 링크](https://www.acmicpc.net/problem/2839)

### 접근 방법 (Thinking Process)

1. 3킬로 5킬로 봉지.
2. 더 큰 5킬로로 최대한 배분 한 후 3킬로로 배분.
3. 나누어 떨어지지 않으면 5킬로에서 한개씩 빼서 다시 보기?

## 2. 코드 설계 (Pseudo Code)

```text
N 입력.
5로 나누기. 몫으로 봉지 카운트 증가. 나머지가 3으로 나누어 떨어지면 몫으로 봉지 카운트 증가.
나누어 떨어지지 않으면 봉지 카운트 차감 후 나머지에 5 더해서 다시 확인.
과정 반복.
봉지 카운트가 음수가 되면 그 즉시 -1 출력 후 종료.
나누어 떨어지면 최종 봉지 카운트 출력.
```

## 5. 최종 코드

```cpp
#include <iostream>
using namespace std;

int main (void) {
    int N = 0;
    cin >> N;
    int c = 0;
    c += N/5;
    int t = c;
    for(int i = 0; i <= t; i++){
        if(((N%5)+(5*i))%3==0) {
            c += ((N%5)+(5*i))/3;
            break;
        } else {
            c--;
            if(c<0) {cout<<-1;return 0;}
        }
    }
    if(c==0) c=-1;
    cout << c << endl;
    return 0;
}
```
